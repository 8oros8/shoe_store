Порядок запуска приложения:
0. Использовать версию Node.js >=10.16.0 <=14.x.x
1. Скачать или склонировать репозиторий в локальные файлы.
2. Перейти в директорию backend
3. Установить зависимости с помощью команды npm install
4. Запустить локальный сервер с помощью команды npm run develop
5. Перейти в директорию frontend
6. Установить зависимости с помощью команды npm install
7. Запустить локальный сервер с помощью команды npm run dev
8. В браузере открыть http://localhost:3000/
9. Наслаждаться!
